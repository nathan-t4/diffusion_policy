# Training Configs
Diffusion policy training configs for pushT environment (adapted from original DP repository)

Configs are processed using [OmegaConf](https://omegaconf.readthedocs.io/en/2.3_branch/)

- `config_v1.yaml`: used for train and evaluate DP model trained using data/pusht_cchi_v1.zarr
- `config_v2.yaml`: used for train and evaluate DP model trained using data/pusht_cchi_v2.zarr
